# 量化交易系统 - 使用指南

## 快速开始（3步）

### 第1步：安装依赖
打开终端，进入项目目录，运行：
```bash
cd D:\workspace\trade_cdx
pip install -r requirements.txt
```

### 第2步：获取数据
运行数据获取脚本，获取A股真实数据：
```bash
python scripts/01_data_analysis_baostock.py
```

### 第3步：运行回测
运行完整回测，测试所有策略：
```bash
python scripts/run_full_backtest.py
```

## 主要脚本说明

### 1. 数据获取
```bash
python scripts/01_data_analysis_baostock.py
```
- 获取平安银行(000001)的真实数据
- 数据保存在 data/cache/ 目录
- 免费，无需注册

### 2. 策略回测
```bash
python scripts/run_full_backtest.py
```
- 测试双均线、RSI、MACD三个策略
- 显示每个策略的收益率、夏普比率等
- 找出最佳策略

### 3. 参数优化
```bash
python scripts/run_optimization.py
```
- 自动搜索最优参数
- 测试不同参数组合
- 推荐最佳参数

### 4. 模拟交易
```bash
python scripts/run_simulation.py
```
- 模拟真实交易环境
- 显示每笔交易详情
- 记录交易历史

### 5. 生成报告
```bash
python scripts/generate_report.py
```
- 生成可视化图表
- 包含净值曲线、回撤图等
- 保存在 reports/ 目录

## 常见问题

### Q1: 运行脚本报错怎么办？
A: 检查是否安装了所有依赖：
```bash
pip install -r requirements.txt
```

### Q2: 数据获取失败怎么办？
A: 检查网络连接，或使用模拟数据：
```bash
python scripts/learn_data_analysis.py
```

### Q3: 如何修改股票代码？
A: 打开脚本，修改 symbol 变量，例如：
```python
symbol = 'sz.000001'  # 平安银行
symbol = 'sh.600519'  # 贵州茅台
symbol = 'sz.000858'  # 五粮液
```

### Q4: 如何修改初始资金？
A: 打开脚本，修改 initial_cash 变量：
```python
initial_cash = 200000  # 20万
initial_cash = 500000  # 50万
```
