## Week 2：Backtrader回测框架

### Day 1-2：Backtrader基础
**目标：** 理解回测框架的工作原理

```python
# 安装
pip install backtrader

# 基本概念
1. Cerebro：回测引擎核心
2. DataFeed：数据源
3. Strategy：交易策略
4. Broker：模拟券商
5. Analyzer：结果分析

# 第一个回测示例
import backtrader as bt
import pandas as pd

# 1. 创建策略
class MyStrategy(bt.Strategy):
    def __init__(self):
        # 计算指标
        self.sma5 = bt.indicators.SimpleMovingAverage(self.data.close, period=5)
        self.sma20 = bt.indicators.SimpleMovingAverage(self.data.close, period=20)
    
    def next(self):
        # 交易逻辑
        if not self.position:  # 没有持仓
            if self.sma5[0] > self.sma20[0]:  # 金叉
                self.buy()  # 买入
        else:  # 有持仓
            if self.sma5[0] < self.sma20[0]:  # 死叉
                self.sell()  # 卖出

# 2. 准备数据
# 假设你已经用tushare获取了数据，保存为csv
data = bt.feeds.PandasData(dataname=pd.read_csv('stock_data.csv', 
                                                  index_col='date', 
                                                  parse_dates=True))

# 3. 创建回测引擎
cerebro = bt.Cerebro()
cerebro.adddata(data)
cerebro.addstrategy(MyStrategy)

# 4. 设置初始资金
cerebro.broker.setcash(200000)  # 20万

# 5. 设置手续费
cerebro.broker.setcommission(commission=0.001)  # 千分之一

# 6. 运行回测
print('初始资金: %.2f' % cerebro.broker.getvalue())
results = cerebro.run()
print('最终资金: %.2f' % cerebro.broker.getvalue())

# 7. 画图
cerebro.plot()
```

class RSIStrategy(bt.Strategy):
    def __init__(self):
        self.rsi = bt.indicators.RSI(self.data.close, period=14)
    
    def next(self):
        if not self.position:
            if self.rsi[0] < 30:  # RSI低于30，超卖
                self.buy()
        else:
            if self.rsi[0] > 70:  # RSI高于70，超买
                self.sell()

# 策略3：MACD策略
class MACDStrategy(bt.Strategy):
    def __init__(self):
        self.macd = bt.indicators.MACD(self.data.close)
        self.crossover = bt.indicators.CrossOver(self.macd.macd, self.macd.signal)
    
    def next(self):
        if not self.position:
            if self.crossover > 0:  # MACD金叉
                self.buy()
        else:
            if self.crossover < 0:  # MACD死叉
                self.sell()
```

# 多因子选股思路
# 1. 选择因子：市盈率、市净率、ROE、营收增长等
# 2. 因子标准化：消除量纲影响
# 3. 因子加权：综合打分
# 4. 选股：选择得分最高的股票

import tushare as ts
import pandas as pd

pro = ts.pro_api('你的token')

# 获取基本面数据
def get_factor_data(date):
    # 获取估值数据
    daily_basic = pro.daily_basic(trade_date=date, 
                                   fields='ts_code,pe_ttm,pb,ps_ttm')
    
    # 获取财务数据
    # 这里简化处理，实际需要更复杂的逻辑
    return daily_basic

# 因子标准化
def normalize_factor(df, col):
    return (df[col] - df[col].mean()) / df[col].std()

# 综合打分
def multi_factor_score(df):
    df['pe_score'] = -normalize_factor(df, 'pe_ttm')  # PE越低越好
    df['pb_score'] = -normalize_factor(df, 'pb')  # PB越低越好
    df['total_score'] = df['pe_score'] * 0.5 + df['pb_score'] * 0.5
    return df.sort_values('total_score', ascending=False)
```

# 常见仓位管理方法

# 方法1：固定仓位
def fixed_position(capital, pct=0.1):
    """每次投入固定比例资金"""
    return capital * pct

# 方法2：凯利公式
def kelly(win_rate, win_loss_ratio):
    """计算最优仓位比例"""
    return win_rate - (1 - win_rate) / win_loss_ratio

# 方法3：ATR仓位管理
def atr_position(capital, atr, risk_pct=0.02):
    """基于波动率的仓位管理"""
    position_size = capital * risk_pct / atr
    return position_size

# 在策略中使用

            self.buy(size=size)

### Day 6-7：风控模块
**目标：** 学会控制风险

```python
# 风控规则
class RiskManager:
    def __init__(self, max_drawdown=0.2, max_position=0.5):
        self.max_drawdown = max_drawdown  # 最大回撤限制
        self.max_position = max_position  # 最大仓位限制
        self.peak_value = 0
    
    def check_drawdown(self, current_value):
        """检查回撤是否超限"""
        if current_value > self.peak_value:
            self.peak_value = current_value
        
        drawdown = (self.peak_value - current_value) / self.peak_value
        if drawdown > self.max_drawdown:
            return False  # 触发风控，停止交易
        return True
    
    def check_position(self, position_value, total_value):
        """检查仓位是否超限"""
        position_pct = position_value / total_value
        return position_pct < self.max_position

# 在策略中集成风控
class MyStrategy(bt.Strategy):
    def __init__(self):
        self.risk_manager = RiskManager(max_drawdown=0.15)
    
    def next(self):
        # 检查风控
        if not self.risk_manager.check_drawdown(self.broker.getvalue()):
            return  # 触发风控，不交易
        
        # 正常交易逻辑
        if self.buy_signal:
            self.buy()
```

## 常见问题

### Q1：Python基础不好怎么办？
**A：** 花3-5天集中学习基础语法，重点掌握：
- 变量、条件、循环
- 函数定义
- 列表、字典操作
- 文件读写

### Q2：回测很好但实盘亏钱？
**A：** 常见原因：
- 过拟合：策略参数太贴合历史数据
- 滑点：实际成交价与预期不同
- 手续费：回测时未考虑
- 市场变化：过去有效的策略现在不一定有效

### Q3：20万资金怎么分配？
**A：** 建议分配：
- 1-2万：小资金实盘测试
- 5-10万：验证有效的策略
- 剩余资金：逐步投入，分散风险

### Q4：需要学多久才能稳定盈利？
**A：** 现实预期：
- 1-3个月：学会基础，能回测策略
- 3-6个月：开始小资金实盘
- 6-12个月：找到适合自己的策略
- 1年以上：逐步稳定盈利
