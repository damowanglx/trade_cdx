# QMT资金门槛问题解决方案

## 问题说明
QMT（迅投）通常有资金门槛要求：
- 部分券商要求50万+
- 部分券商要求100万+
- 你的20万资金可能不够

## 解决方案

### 方案1：换券商（推荐）

有些券商QMT门槛较低，可以咨询：

| 券商 | QMT门槛 | 联系方式 |
|------|---------|----------|
| 国金证券 | 需要咨询 | 客户经理 |
| 华鑫证券 | 低门槛 | 需要咨询 |
| 国盛证券 | 低门槛 | 需要咨询 |
| 东方财富 | 低门槛 | 需要咨询 |

**操作步骤：**
1. 联系上述券商客户经理
2. 询问QMT开通门槛
3. 如果门槛合适，开通账户
4. 转移资金到新账户

### 方案2：使用easytrader（免费，无门槛）

**优点：**
- 完全免费
- 无资金门槛
- 支持多家券商

**缺点：**
- 模拟键鼠操作，不太稳定
- 需要券商客户端运行
- 可能违反券商规定

**支持的券商：**
- 华泰证券
- 国金证券
- 银河证券
- 其他...

### 方案3：使用聚宽/米筐（在线平台）

**优点：**
- 无需本地部署
- 免费使用
- 数据丰富

**缺点：**
- 需要学习新平台
- 实盘需要付费
- 策略需要迁移

**平台：**
- 聚宽：https://www.joinquant.com/
- 米筐：https://www.ricequant.com/
- 优矿：https://uqer.datayes.com/

### 方案4：等待资金达标

**优点：**
- 可以继续学习
- 积累更多经验
- 策略更成熟

**缺点：**
- 需要时间
- 可能错过机会

### 方案5：使用PTrade

**优点：**
- 部分券商门槛较低
- 功能类似QMT

**缺点：**
- 需要重新学习
- 券商支持有限

## easytrader方案详细步骤

### 1. 安装easytrader
```bash
pip install easytrader
```

### 2. 选择券商
- 华泰证券（推荐）
- 国金证券
- 银河证券

### 3. 配置券商客户端
- 下载并安装券商客户端
- 登录账户
- 保持客户端运行

### 4. 使用easytrader
```python
import easytrader

# 连接券商
user = easytrader.use('ths')  # 同花顺
# user = easytrader.use('ht')  # 华泰
# user = easytrader.use('gj')  # 国金

# 登录
user.connect(r'C:\国金证券\国金证券QMT交易端\bin.x64\XtMiniQmt.exe')

# 获取持仓
user.position

# 买入
user.buy('000001', price=10.5, amount=1000)

# 卖出
user.sell('000001', price=11.0, amount=1000)
```
