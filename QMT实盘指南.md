## 下一步操作

### 1. 复制策略文件到QMT

将策略文件复制到QMT的python目录：

```
D:\workspace\trade_cdx\qmt_strategy_rsi.py  →  D:\国金QMT\国金证券QMT交易端\python\
D:\workspace\trade_cdx\qmt_strategy_macd.py →  D:\国金QMT\国金证券QMT交易端\python\
```

### 2. 在QMT中加载策略

1. 打开QMT交易端
2. 进入"量化交易"或"策略交易"模块
3. 选择"导入策略"
4. 选择复制过去的策略文件
5. 配置交易股票和资金

### 3. 策略参数说明

#### RSI策略（推荐）
- RSI周期：21
- 超卖阈值：30（买入信号）
- 超买阈值：70（卖出信号）
- 仓位比例：90%

#### MACD策略
- 快速EMA：12
- 慢速EMA：20
- 信号线：7
- 仓位比例：90%

### 4. 风险控制

- **止损**：建议设置8%止损
- **止盈**：建议设置20%止盈
- **仓位**：单只股票不超过50%仓位
- **分散**：同时持有3-5只股票

### 5. 测试流程

1. **模拟盘测试**（1-2周）
   - 在QMT中开启模拟交易
   - 观察策略信号是否正常
   - 检查下单是否成功

2. **小资金实盘**（1-2万）
   - 验证策略在真实市场的表现
   - 记录每笔交易
   - 分析收益和风险

3. **逐步增加资金**
   - 验证稳定后逐步增加
   - 不要一次性投入全部资金

## 注意事项

1. **市场风险**：量化交易存在风险，回测结果不代表未来收益
2. **策略优化**：市场环境变化时需要调整参数
3. **监控交易**：定期检查策略运行状态
4. **备份策略**：保存好策略代码和参数
