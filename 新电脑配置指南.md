# 量化交易系统 - 新电脑配置指南

## 一、环境准备

### 1. 安装Python（3.8+）
- 下载地址：https://www.python.org/downloads/
- 安装时勾选"Add Python to PATH"

### 2. 安装依赖包
```bash
cd trade_cdx
pip install -r requirements.txt
```

### 3. 安装QMT（国金证券）
- 从国金证券官网下载QMT交易端
- 安装到D盘（建议路径：D:\国金QMT）

---

## 二、项目配置

### 1. 数据获取
```bash
# 获取A股数据（约30分钟）
python scripts/fetch_all_stocks_v2.py

# 获取沪深300基准数据
python scripts/fetch_hs300.py
```

### 2. 邮件报警配置（可选）
编辑 `monitor/email_alert.py`，填写邮箱信息：
```python
alert.configure(
    sender_email='your_email@qq.com',
    sender_password='your_authorization_code',  # QQ邮箱授权码
    receiver_email='receiver@email.com'
)
```

### 3. QMT策略配置
将策略文件复制到QMT目录：
```bash
copy qmt_strategy_rsi_complete.py D:\国金QMT\国金证券QMT交易端\python\
```

## 五、注意事项

### 1. 数据文件
- `data/all_stocks/` 目录下的CSV文件不会提交到Git
- 需要重新运行 `python scripts/fetch_all_stocks_v2.py` 获取数据
- 数据获取约需30-60分钟

### 2. QMT路径
- 如果QMT安装路径不同，需要修改策略文件中的路径
- 默认路径：`D:\国金QMT\国金证券QMT交易端\`

### 3. 邮箱配置
- QQ邮箱需要开启SMTP服务
- 获取授权码（不是QQ密码）
- 配置文件中的邮箱信息不会提交到Git

### 4. Python版本
- 建议使用Python 3.8-3.12
- Python 3.14可能有兼容性问题

## 七、常见问题

### Q1: 数据获取失败
- 检查网络连接
- 尝试使用VPN或代理
- 分批获取数据

### Q2: QMT连接失败
- 确保QMT交易端已启动
- 检查QMT路径是否正确
- 确认已登录QMT账户

### Q3: 依赖包安装失败
- 使用国内镜像源：
  ```bash
  pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
  ```

### Q4: 策略运行异常
- 检查股票代码格式（如：sz.000001）
- 确认数据文件存在
- 查看日志文件
